# 人体动作识别系统 python413

## 项目概述

人体动作识别系统是一个基于Python开发的简易型应用，主要用于识别人体动作。结合了Tkinter图形界面、OpenCV图像处理和MediaPipe姿态估计技术，本系统为用户提供了一个实时检测并反馈人体动作的平台。

## 技术栈

- **图形界面：** Tkinter
- **图像处理：** OpenCV
- **姿态估计：** MediaPipe

## 功能模块

- 实时人体动作识别
- 视频和图片上传分析
- 关键点检测（肩膀、臀部、膝盖、脚踝等）
- 动作类型识别（坐姿、站立、单腿站立、挥手、拍手、行走、跑步、慢跑、摔倒检测等）

## 系统角色

- **用户：** 通过界面与系统交互，上传视频或图片，启动摄像头实时检测。

## 运行环境

- Python运行环境
- 需要安装的依赖库：Tkinter、OpenCV、MediaPipe

## 部署步骤

1. 确保Python环境已正确安装。
2. 安装必要的依赖库。
3. 克隆或下载项目代码。
4. 运行系统启动脚本。

## 目录结构

```
项目目录/
│
├── main.py               # 系统主入口
├── gui.py                # 图形界面相关代码
├── mediapipe_pose.py     # MediaPipe姿态估计处理
├── video_processing.py   # 视频处理相关代码
└── utils.py              # 实用工具函数
```

## 核心亮点

- **简洁直观的界面设计：** 界面简洁，操作便捷，满足不同用户的操作需求。
- **多线程技术：** 界面更新与视频处理分离，保障系统运行的流畅性。
- **可扩展性：** 基础简单示例代码，便于后续开发与功能扩展。
